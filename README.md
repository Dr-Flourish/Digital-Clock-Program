Digital Clock

This is a simple digital clock built using HTML, CSS, and JavaScript. It displays the current time and updates automatically every second in real time.

Features

Real-time display of hours, minutes, and seconds

Automatically updates every second

Clean and simple user interface

Lightweight and easy to understand

Technologies Used

HTML – creates the structure of the clock

CSS – styles the clock layout and appearance

JavaScript – retrieves the current time and updates the display

How It Works

JavaScript uses the Date object to get the current time and updates the clock every second using setInterval().

How to Use

Download or clone the repository

Open the digitalClockProgram.html file in any web browser

The digital clock will start running automatically

Project Purpose

This project was created to practise JavaScript timing functions and DOM manipulation.
